# 巡检系统前端

这是一个基于微信小程序开发的巡检系统前端应用。

## 功能特点

- 用户登录与认证
- 巡检记录管理
- 事件记录与追踪
- 语音录入功能
- 图片上传功能
- 离线工作模式

## 技术栈

- 微信小程序原生开发
- WeUI 组件库
- 微信开放能力（录音、拍照等）

## 开发环境

1. 安装微信开发者工具
2. 导入项目
3. 配置服务器地址
4. 开始开发

## 项目结构

```
inspection-frontend/
├── components/      # 自定义组件
├── images/          # 图片资源
├── pages/          # 页面文件
├── services/       # 服务层
├── utils/          # 工具函数
└── app.json        # 全局配置
```
